1. restaurant_recommendations folder contains the .exe file of the gui.

2. Open gui folder in matlab and run gui.m file or simply type gui in matlab.

3. costfunction.m computes the cost and gradient.

4. main.m file can be used to run the algorithm without using gui i.e. in matlab command area.

5. checkcostfunction.m and computeNumericalGradient.m is to check whether the partial gradient is being calculated
   correctly.

6. fmincg.m for optimisation using advanced Wolfe-Powell technique (available in mathworks.com)

7. more details are mentioned as comment in each file.
